## 高低通滤波

### 光字

| 滤波器 | 现象                   |
| ------ | ---------------------- |
| 全通   | 亮，有网格             |
| 低通   | 极暗，无网格，难以分辨 |
| 高通   | 暗，无网格，模糊       |

### 十字

| 滤波器 | 现象                     |
